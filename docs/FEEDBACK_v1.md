## グラフ設計方針（調査結果 + 04/10 議論）

### 調査結論
- Obsidian / Logseq / Roam の全体グラフは「見て楽しいけど使えない」と広く批評されている
- 実用的なのは **Heptabaseのキャンバス**（手動配置+位置記憶）と **Obsidianのローカルグラフ**（1ノード起点の局所表示）
- 動的な物理シミュレーション（ふわふわ浮く）は見世物であって実用ではない

### 確定方針
1. **フワッと浮くアニメーションなし** — d3 force simulationは初回配置にのみ使用。配置後は静止
2. **ドラッグで位置調整 → 位置を記憶** — ユーザーが動かしたらそこに固定。次回も同じ位置
3. **タップ → 滑らかズーム → ローカルグラフ化** — タップしたノード中心に、直接繋がる外部知識が放射状に展開
4. **展開したノードも含めて繋がりを作れる** — 外部知識ノード同士、外部知識↔メモノード間の紐付け
5. **ノード内テキスト: 7-8文字**（円の直径60-70px、9px font）。4文字→7-8文字に拡大。15文字はスマホで円に収まらないため不採用
6. **ノードサイズに意味** — 接続数が多いほど大きい
7. **接続線の視覚区別** — combination（太い実線）vs external_knowledge（細い破線）
8. **ピンチアウトで全体グラフに戻る**

### 未設計（次セッション）
- 深掘りチャットの結果をノードに転記する仕組み（「ノートに追記」or「新ノードとして生成」）
- ノード編集履歴の可視化
- ローカルグラフの展開深度（1段 or 2段）

## ホーム画面設計方針（調査結果 + 04/10 議論）

### 調査結論
- 競合（Voicenotes, Napkin, Apple Voice Memos, Audionotes, Otter）は全て「ホーム＝一覧＋録音ボタン」
- ランチャー画面（4カード等）を採用しているアプリはゼロ
- ミニマル系（Gravity Notes, Napkin, Reflect）は「ホームで過去を見せない」思想
- Napkinの「Daily Flow」（AIが過去メモを再浮上）が最もideamemoに近い

### 確定方針
- **ホーム画面は究極にシンプル** — 偉人の言葉 or AIが再浮上させた自分の気づき1つ。それだけ
- **メモ一覧ページは作らない** — グラフのノードが一覧の代わり
- **フォルダ機能は消す** — v1に不要。20件でフォルダは過剰（Napkin, Gravity Notes, Reflectもフォルダなし）
- **フッター3タブ維持** — ホーム / 録音 / グラフ
- **ヘッダー** — 左ロゴ / 右マイページ

## 04/11 セッション — 具体⇔抽象の可視化 + 録音結果画面 + ノードラベル

### 核心の方向性
ideamemoの価値は「具体→抽象→他業界の具体」の往復。この変換プロセスをUIで見せることが重要。

### 録音結果画面の確定方針（v5）
- **画面遷移なし** — 録音画面と結果画面は同一。録音停止後に同じ画面内で結果が展開
- **文字起こし+抽象原則はすぐ出る**（構造化は高速）
- **外部知識検索中はローディング+名言表示**
- **3件揃ったらローディング終了** → 名言消える → カードが上から順にフェードイン（各300-500ms間隔）
- 「↓」矢印で具体→抽象の変換プロセスが可視化される

#### 画面の流れ
1. 録音停止 → 文字起こし+抽象原則が即座に表示
2. その下に名言+「同じ構造を探しています」+3ドットpulse
3. 3件揃った → 名言フェードアウト →「同じ構造が見つかりました」
4. カード①フェードイン → 300-500ms → カード②フェードイン → 300-500ms → カード③フェードイン

#### SSEイベント対応
- `transcription + structured` → Phase 1表示（文字起こし+抽象原則+名言ローディング）
- `connection` ×3件到達 → Phase 2（名言消える→3カード順次フェードイン）
- `done` → 完了

### ノードラベルの変更方針（v5）
- **デフォルトを抽象原則に変更**（具体summary → 抽象graph_label）
- `graph_label` フィールドを types.ts に追加（7文字以内の短縮抽象、構造化時に生成）
- タップで具体に降りる構造
- 例: 「通勤ラッシュ...」→「合理性の罠」、「会議の沈黙問...」→「最初の一歩」

### メタ分析 / スマートフィルタ（v5.5候補）
- グラフ画面で「思考の傾向」を表示（「集団行動の構造」に関するメモが多い、等）
- テーマ別にノードをハイライト
- フィルタ3軸（time/connectivity/source）は15ノード超過まで非表示

### 実装順序
